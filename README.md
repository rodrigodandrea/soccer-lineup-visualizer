# ⚽ Soccer Lineup Visualizer

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://soccer-lineup-visualizer.streamlit.app/)

A professional tactical board for soccer coaches and enthusiasts. This application allows users to create, visualize, and manage team lineups with a clean, data-driven interface.

[Image of a soccer tactical pitch layout showing player positions and coordinates]

## 🌟 Key Features

* **Bilingual Interface:** Toggle between **English** and **Spanish** seamlessly.
* **Dynamic Formations:** Support for classic layouts like 4-4-2, 4-3-3, 3-5-2, and modern variations like 4-1-2-2-1.
* **Smart Positioning:** Automatically maps player names to specific tactical coordinates (x, y) on a professional-grade pitch.
* **Squad Management:** Dedicated sections for both **Starting XI** and **Substitutes** (The Bench).
* **Visual Analytics Style:** High-contrast pitch design based on professional sports analytics standards (StatsBomb style).

## 🛠️ Tech Stack

* **Python 3.12**
* **Streamlit:** For the interactive web interface.
* **mplsoccer:** Professional library for soccer pitch visualizations.
* **Matplotlib:** For high-quality graphic rendering.

## 🚀 How to Use

1.  Access the **Live Demo** via the badge at the top.
2.  Select your preferred **Language** and **Formation** in the sidebar.
3.  Enter the names of your players in the designated position fields (GK, RB, CDM, etc.).
4.  Add your **Substitutes** and assign them a primary role.
5.  Click on **"Generate Tactical View"** to render
